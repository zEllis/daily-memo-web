# DailyMemo Web 管理界面

这是 DailyMemo 的 Web 管理界面，用于管理每日笔记和提醒。

## 功能

- 📝 创建、编辑、删除笔记
- ⏰ 设置提醒时间
- 🔍 搜索和筛选笔记
- 📱 立即推送笔记到微信

## 使用方法

1. 访问网站
2. 输入 API Token
3. 开始管理你的笔记

## 技术栈

- 纯 HTML + JavaScript
- 无需后端服务器
- 连接到 AWS Lambda API

## 安全说明

- API Token 保存在浏览器本地
- 不会上传到任何服务器
- 请妥善保管你的 Token

## 部署

本项目可以部署到：
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages
